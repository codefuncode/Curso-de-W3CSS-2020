CSS receptivo moderno

Igualdad para todos los navegadores: Chrome. Firefox Edge. ES DECIR. Safari. Ópera.

Igualdad para todos los dispositivos: escritorio. Ordenador portátil. Tableta. Móvil.

CSS estándar solamente (sin jQuery o biblioteca de JavaScript).


W3.CSS es un marco CSS

W3.CSS es un marco CSS moderno con soporte para escritorio, tableta y diseño móvil de forma predeterminada.

W3.CSS es más pequeño y más rápido que los marcos CSS similares.

W3.CSS fue diseñado para ser una alternativa de alta calidad a Bootstrap.

Fue diseñado para ser más pequeño, más rápido, más fácil de aprender y más fácil de usar.

También fue diseñado para ser independiente de jQuery o cualquier otra biblioteca de JavaScript.

</html>
